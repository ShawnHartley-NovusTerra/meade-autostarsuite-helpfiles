Telescope menu commands

The Telescope menu offers the following commands:

Protocol Selects the protocol to use to communicate with your telescope.
Set Telescope Fields Selects the size and shape of both the Guide scope and Main scope fields. The sizes of the fields may also be chosen by selecting from a list of popular Meade equipment.
Hand Control Panel Displays the Telescope Hand Control Panel that allows slewing and guiding, selection of slewing speeds, and focusing controls.
Scope Setup/Shutdown This optional menu is display only when Autostar Suite is communicating with the telescope. It allows the easy initialization or shurdown and the setting and reading of various time and location values of the LX200 series telescopes.
Auto Track Toggles the state of the Auto Track feature. When enabled, the Starmap is automatically redrawn to show the telescope position as the scope moves across the sky.
Communications Provides access to the configuration and testing of communciation between the computer and telescope:

Start Network Server

CommPort Setup

Test.
Dome Control Opens or selects Meades Observatory Control module to allow you to coordinate dome motion with telescope motion and remotely control other observatory structures
Autostar Updater Opens or selects the Autostar Updater to install new firmware, download tours, asteroids, comets, and satellites to your telescope.
PEC Tool Opens or selects the Autostar PEC Tool. The PEC Tool allows interactive tuning of the Smart Drive correction parameters in the Autostar II Family of telelscopes to your telescope.
