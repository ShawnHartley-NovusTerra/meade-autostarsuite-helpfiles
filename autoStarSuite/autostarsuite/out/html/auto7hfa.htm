Image menu commands

The Image menu offers the following commands:

LPI Imaging Starts the Envisage application module to control an LPI Imaging session.
DSI Imaging Starts the Envisage application module to control a DSI or DSI II Imaging session.
Pictor Camera Imaging Starts the PictorView application module to control a Pictor 208XT, 216XT 416XTE or 1616XTE imaging session
Pictor Autoguiding Stars the Pictor Autoguiding program to utilize a Pictor 208XT or Pictor 216XT as an computer controlled autoguider.
Remote Envisage Starts the Enviage remote control module to allow you to remotely operate a DSI, DSI II or LPI    
Image Processing Starts the Autostar Image Processing module to perform advanced image processing and analysis
