Starmap menu commands

The Starmap menu offers the following commands, which control the appearance of the starmap display:

Map ON/OFF Toggles the Starmap on and off.
Previous View Resets the Starmap values and options to their previous states and redraws the map.
Redraw Redraws the current Starmap. Useful when the display becomes cluttered by adding new options repeatedly.
Animate Starmap Animates the planets, asteroids, comets, and user planetoids. The starting time and step times can varied. Also, the individual object classes can be selected as needed.
Viewpoint Allows the coordinates and viewing angle of the Starmap to be selected. The coordinates define the center of the Starmap.
Lock AzEl When selected, the star map center coordinate remains centered on an Azimuth and Elevation as seen from the observer;s location.
Zenith Up When set, this option rotates the Starmap so that zenith is towards the top of the display. This is the perspective that a gound based observer, or Alt/Az mounted telescope will see. When not set, the celestial pole is at the top of the display. This is the persective seen by an equatorially mounted telescope.
Object Selection Allows any object (Messier, NGC, IC, SAO, Constellations, etc.) to be selected and displayed in the center of the Starmap. A description of each object is also available for most objects.
Magnitudes Sets the brightest and dimmest object magnitudes. Only those objects with magnitudes between these values will be displayed. Default values are -5 and 22. Alternatively, you can enable automatic magnitude range selection. When enabled, the magnitude range is automatically set, based upon the field of view, to keep a reasonable number of objected displayed.
Mirror Allows the orientation of the Starmap to be adjusted to account for any image reversals that are due to the telescope optics. The image can be flipped about both the vertical and horizontal axes.
Zoom Sets the current Field of View for the Starmap. The value can be entered directly or rapidly chosen from the Sky Coverage buttons.
Hubble Guide Stars Enables or disables the optional Hubble Guide Star catalog. The drive designation of the CD-ROM must be set before the catalog can be accessed.
Invert Screen Colors Causes the Starmap display to be drawn with a white background and different object colors. Useful for those who prefer a 'chart like' display.
Precision Selects the precision to which the Ephemeral objects are calculated. Lower precision will allow the objects to be drawn more quickly. Geocentric calculates position relative to the center of the Earth, while Topocentric is relative to current location coordinates on the Earth's surface.
Projection Allows selection of either Azimuthal Equidistant or Spherical projections. Azimuthal Eq. is useful for full sky views. Spherical is better for smaller fields of view.
Star Contrast & Brightness Allows the contrast and brightness of the stars to be varied to more closely match the view of the actual sky.
Reset Parameters Resets many of the important Starmap options to a default state.
