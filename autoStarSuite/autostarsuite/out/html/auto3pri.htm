Set Time & Date

Setting the current time and date can be accomplished in two different manners. You may enter either the standard (Gregorian) time and date values or you may enter the full Julian date. Either method is equally valid. Once you specify a time and date, the internal Autostar Suite clock is frozen at your desired settings. The internal clock runs only when you are using the system clock (either from the PC or the observatory). To start the system clock, select either Gregorian or Julian from the Set Time & Date menu and push the Use System Time & Date button.

The entered value may also be used as the current Dynamical Time by selecting the Set As Dynamical Time button.
