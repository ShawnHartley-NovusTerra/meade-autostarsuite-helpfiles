Time Menu commands

The Time menu offers the following commands, which provide you the capability to choose any time from 4713 B.C. to 9999 A.D. at any location on Earth:

Time Display Allows you to continuously display a variety of important times.
Set Time & Date Allow you to set the current time and day in either the Gregorian format (hh:mm:ss yyyy/mm/dd) or
Julian format (dddddddd.ffffffff).

The new time remains frozen until the Use System Time button is selected.
Set Location Allows you to select the current viewing position. The coordinates may be entered manually or automatically by selecting from a list of observatories or cities.
Save Parameters Saves the current location parameters.
