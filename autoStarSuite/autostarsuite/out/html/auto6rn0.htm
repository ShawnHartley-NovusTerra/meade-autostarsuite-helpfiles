File menu commands

The File menu offers the following commands:

Open Opens an existing image.
Close Closes an opened image.

Save Map Save current starmap as a BMP file on disk.

Print Prints the starmap in color or black and white.
Print Setup Selects a printer and printer connection.
Page Setup Selects page options such as, borders, margins, etc..
Set Custom Catalogs
 Selects the additional files to be displayed as objects and planetoids.
Compile Compiles binary versions of the various objects databases.
Build Tables Builds the binary tables used by Autostar Suite.
Preferences Allows selection of various display options.
Exit Exits Autostar Suite.
