Setup Status Bar

The Setup Status Bar dialog allow you to selectively enable or disable a number of useful status fields. The following tables describes the various fields:

FieldDescription

DateDisplay the current Gregorian date.

Local TimeDisplays the current local time or the current set time.

Mean Solar TimeDisplays the Mean Solar time at the current location.

Universal TimeDisplay Universal time.

Local Sidereal TimeDisplays the Sidereal time at the current location.

R.A. / x,yDisplays the Right Ascension value of the mouse position when in the Starmap display, or the x and y coordinates when in an Image display.

Dec. / DnDisplays the Declination value of the mouse position when in the Starmap display, or the pixel value under the cursor when in an Image display. The pixel value show both the 8 bit and 16 bit values simultaneously.

Alt. / Dist.Displays the Altitude value of the mouse position when in the Starmap display, or the Distance value if the Set Distance Reference is enabled.

Az. / AngleDisplays the Azimuth value of the mouse position when in the Starmap display, or the Angle value if the Set Distance Reference is enabled in an Image display.

NOTE: If all of the fields are enabled, you may not see every field if your screen resolution is to low. Try selecting fewer fields or change to a higher screen resolution.
