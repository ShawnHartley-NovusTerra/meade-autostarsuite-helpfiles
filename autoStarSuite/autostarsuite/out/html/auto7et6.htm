Ephemeris menu commands

The Ephemeris menu offers the following commands, which provide you information on the ephemeral (changing) solar system objects:

Sun, Moon, etc. Displays a variety of important values on the selected object. The precision of these values is determined by the values set in the Starmap:Precision menu

An Ephemerides (a list of position values over time) can also be generated and printed.
Import Active Asteroids and Comets Calendar Allows you to import comet and asteroid data, in Autostar format from the most recent Internet source
Moon Calendar Display a monthly calendar show the phase of the moon at 0hr. Universal Time on each day.
Eclipse Calendars Displays work maps depicting the path and duruation of total and annular eclipsed through the year 2020.
Show Current Values Displays a variety of high precision values that are used for calculating the Ephemeris values. The values can be displayed in either dd:mm:ss or dd.fffffff formats.
