Startup Connect->Connect & Retrieve (Options menu)

When this option is toggled on, the program automatically attempts to connect to the Autostar Handbox and retrieve data every time the program is launched. In order to successfully connect to the handbox, the handbox must be connected to the PC with a P/N 07047 LX200 Interface Cable, #505 or #506 Connector Set and powered on.

Turning this option on automatically displays the contents of the handbox, but it may slow program performance.
