File menu commands

The File menu offers the following commands:

New Creates a new document.
Open Opens an existing document.
Save Saves an opened library document using the same file name.
Save As Saves an opened library document to a specified file name.
Save Hbx Data Saves the data from the Handbox List View to a specified file name.
Load Hbx Data Loads an existing Handbox List View document.
Restore Data Restores data to the Handbox List View that was automatically backed up after the last Send or Retrieve command was completed.
Import Imports data from a file other than an Autostar Update Data (*.aud) file.
Get Object Data from WWW Retrieves Object data from the web and puts it into the library.
1,2,3,4 Opens one of the 4 most recent library documents.
Exit Exits Autostar Update.
