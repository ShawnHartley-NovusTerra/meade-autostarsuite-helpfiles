New Object (Edit menu)

Creates a new object in the last selected list view. Depending on the type of object active in the list view, a dialog box displays showing the appropriate blank fields. Enter the data for all the fields, being careful to enter values within the ranges of the field. Values that are outside the allowable ranges or left blank are assigned default values.

Right-mouse-clicking a selected object(s) from either list view may also access this command.
