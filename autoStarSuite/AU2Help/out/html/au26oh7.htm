Transfer All Objects To Library (Dialog Box Button)

Clicking this button allows all objects in the Handbox List View to be copied to the Library List View. A dialog box will appear allowing for selection of object types to be copied. Click the check boxes next to the types of objects you wish to copy to the Library List View, or click "ALL Object Types" to copy the entire Handbox List View. Note that any objects of the selected type that are already in the Library List View will be deleted, and then replaced with the copied objects.
