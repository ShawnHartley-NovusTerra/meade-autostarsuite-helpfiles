How To Transfer Objects To And From The Handbox And Library Collection

To select this function, you must be in Advanced View mode. The Advanced View window displays the Library List View on the left and the Handbox List View on the right.

The Library List View contains the master list of objects Loaded or Imported from your local hard disk. The Handbox List View contains the data that will be Sent to the Autostar Handbox, or that have just been Retrieved from the Autostar Handbox.

To transfer objects from one List View to the other:

1.Select the appropriate object type by pressing one of the 6 buttons (Asteroids, Satellites, Comets, etc.) at the top of either list. Pressing one button from either list changes the displayed object type in BOTH lists. Note: All data types are stored in the Autostar Update program at all times, but the list view only displays one object type at a time.

2.Select the objects from one list view that you wish to copy to the other list view. You may select one at a time by clicking on an object with the mouse, or you may select multiple objects by holding down the shift key or CTRL key. You may only select objects of the same type, however.

3.Either drag the selected items from one list view to the other by holding down the mouse button and releasing it when the objects are over the destination list view, OR click on the Copy ObjectsButton (labeled with a "->" or "<-" arrow).

Note: Any objects copied to the Handbox List View are not yet in the handbox. You may sort the list by clicking on the desired column heading, edit objects, delete objects, or create new objects.

When the Handbox List View contains all the objects you wish to transfer to the handbox:

1.Connect the Handbox to the PC using the appropriate cable kit.

2.Turn the telescope power on.

3.Click on the Send Data button in the Autostar Commands button group located in the upper right of the window. It may take several minutes to send the data, depending on how much data is being transferred. Once the data is sent, it is automatically saved to a backup file, so you can undo any changes you by choosing the File Menu Item Restore Data.

Caution: Sending data to the handbox erases all the objects already in the handbox, so if you do not wish to permanently lose this data, Retrieve it first, Save it to the hard disk, and then proceed as above.
