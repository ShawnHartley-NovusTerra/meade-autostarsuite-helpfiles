List View Status Bar

At the bottom of the Library List View and the Handbox List View is a status bar providing important information about each of those list views.

IndicatorDescription

Object CountThe number of objects of the selected object type and the number of objects in the entire list.

MemoryThe Library List View status bar displays the total memory required for the library object collection. The Handbox List View status bar displays the remaining memory in the handbox for additional objects.

It is important not to exceed the memory capacity of the handbox when building an object collection in the Handbox List View. The available memory reported in the status bar is calculated by subtracting the sum of the memory required for all the objects in the collection from the total memory available in the handbox (which differs depending on the Autostar Model Number). If you attempt to transfer more objects than the memory of the handbox can hold, the status bar displays a negative number in red, and prevents you from issuing the Send Data command.
