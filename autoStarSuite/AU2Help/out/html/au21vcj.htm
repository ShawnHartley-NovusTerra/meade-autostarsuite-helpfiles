Transfer Buttons

To transfer or copy objects from one List View to the other, use one of the three pairs of transfer buttons:

Selected

Only the selected object type indicated by the depressed object button above either list view is affected by these commands.

-> copies only the objects that have been selected (highlighted) in the Library List View to the Handbox List View.

<- copies only the objects that have been selected (highlighted) in the Handbox List View to the Library List View.

All

All object types, whether visible or not, are affected by these commands.

>> presents a dialog box to select the object type to copy from the Library List View to the Handbox List View.

<< copies every object in the Handbox List View to the Library List View.

Refresh

Only the selected object type indicated by the depressed object button above either list view is affected by these commands. Use these commands to update your ephemerid data for objects that already exist in your handbox.

> only copies objects that already exist in the Handbox List View from the Library List View.

< only copies objects that already exist in the Library List View from the Handbox List View.
