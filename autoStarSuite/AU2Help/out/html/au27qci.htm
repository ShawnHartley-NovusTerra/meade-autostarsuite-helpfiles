Upgrade Autostar Software Now (Dialog Box Button)

Use this button to:

1)Download the latest version of your handbox’s software from Meade Instruments Corporation’s web site and upgrade your handbox; OR

2)Download the latest version of your handbox’s software from Meade Instruments Corporation’s web site and save the file to a local disk drive for a future upgrade of the handbox; OR

3)Upgrade the handbox using a file previously saved on a local disk drive.

No matter which function you wish to perform, clicking on this button starts a process which automatically guides you through the procedure with a series of dialog boxes.

For more information on upgrading your handbox, see How To Upgrade your handbox to a different revision of the software.
