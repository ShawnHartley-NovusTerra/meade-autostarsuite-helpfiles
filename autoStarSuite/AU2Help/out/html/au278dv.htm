Startup Connect->Connect Only(Options menu)

When this option is toggled on, the program automatically attempts a connection to the Autostar Handbox every time the program is launched. In order to successfully connect to the handbox, the handbox must be connected to the PC with a P/N 07047 LX200 Interface Cable, #505 or #506 Connector Kit and powered on.

Turning this option on automatically displays the status of available memory and the number of objects stored in the handbox, but it may slow program performance.
