Restore Data command (File menu)

Every time you Send Data to the hand box a backup of what was sent is saved to the hard disk. Every time you Retrieve Data from the Autostar Handbox, a backup file of what was received is saved to the hard disk.

To restore this list and erase any changes you have made since the backup file was created, choose "Restore Data" from the "File" menu. This feature can be used to restore the last handbox data instead of retrieving it from the handbox.
