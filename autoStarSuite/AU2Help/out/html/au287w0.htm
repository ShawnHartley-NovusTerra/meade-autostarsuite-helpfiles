Transfer All Objects To Handbox (Dialog Box Button)

Clicking this button allows all objects in the Library List View to be copied to the Handbox List View. A dialog box will appear allowing for selection of object types to be copied. Click the check boxes next to the types of objects you wish to copy to the Handbox List View, or click "ALL Object Types" to copy the entire Library List View. Note that any objects of the selected type that are already in the Handbox List View will be deleted, and then replaced with the copied objects.
