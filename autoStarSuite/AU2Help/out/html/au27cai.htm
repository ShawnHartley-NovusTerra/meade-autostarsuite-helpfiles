Verify Downloads (Options menu)

When this option is toggled on, the data sent to the handbox is verified after transfer. Every time the Upgrade Autostar Software Now button or the Send Data button is clicked, a full scan of the handbox memory isperformed to compare it against the original data and ensure that the data was correctly transferred.

Note: This option DOUBLES the amount of time it takes to send data to the handbox. For example, if it takes 20 minutes to upgrade the handbox code normally, it will take 40 minutes to upgrade it with this option turned on. Do not use this option unless you feel you are having problems transferring data.
