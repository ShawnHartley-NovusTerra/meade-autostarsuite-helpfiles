Autostar Commands (Button Group)

This button group contains the 4 commands for communicating with the Autostar handbox. Before using one of these commands, make sure the handbox is properly connected to the PC using a P/N 07047 LX-200 Interface Cable, #505 or #506 Connector Cable Kit and that the Autostar is powered on.

Connect Establishes connection between the software and the handbox.
Retrieve Data Copies objects from the Autostar handbox to the Handbox List View.
Send Data Copies objects from the Handbox List View to the Autostar handbox.
Real Time Mode Toggles operation of Autostar Update between normal and Real Time Mode.
