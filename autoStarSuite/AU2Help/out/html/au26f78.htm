How To Convert Your Files From The Previous Version Of Autostar Update

The object data in previous versions of Autostar Update were stored as separate files; one for each body type. In order to use this data it must be imported into the new library. As each file is imported it will be added to the current library. To Import follow these steps

1.From the menu select File->Import

2.Pull down <Files of type> and select <ROM files>

3.Select the directory that the old files were stored in by pulling down <Look in> and selecting <Local Disk (C:)> then double click on each of these directories in this order: Program Files->Meade->Autostar->Autostar Update->Ephemerides. This is the standard install directory. If you installed in a different directory then select the Ephemerides directory under your install directory.

4.Select the file LibAsteroids.ROM this is obviously the asteroids data.

5.Pull down Object Type and select Asteroid.

6.Click Import. The data will be added to your library.

7.Repeat steps 1 – 6 for each of the other Object Types.

8.Save the library by selecting File->Save, or File->Save As and select a name and location that you will remember. I suggest My Documents for the location type Library as the name.

9.Remove the old program by going to Control Panel and selecting Add Remove Programs and then select Autostar Update and click Remove. Do not remove ASU which is this program.
