Advanced View vs. Basic View (Dialog Box Button)

For novice users of Autostar who simply wish to upgrade their handbox to a newer version of the handbox software, keep the Autostar Update program in "BASIC" view, which provides only one option - Upgrade Autostar Software Now.

For advanced users of Autostar who wish to manipulate the objects stored in the handbox, such as Asteroids, Comets, Satellites, Landmarks, Tours and User Objects, click on the ">> Advanced Functions>>" button to expand the window size. The advanced window displays the Autostar Commands button group as well as the Handbox List View and the Library List View.
