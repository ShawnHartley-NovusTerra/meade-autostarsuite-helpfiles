User Objects (Object Data Type)

A user object is ANY celestial object that a user defines with the Autostar Handbox. The location of this object are recorded in celestial coordinates, so whenever the telescope is astronomically aligned, the user may return to that object. This is useful for those wishing to observe celestial objects that are not currently defined in the Autostar’s database, such as newly-found comets or asteroids.

The Autostar II handbox also contains Custom User Objects, which have user-defined fields.
