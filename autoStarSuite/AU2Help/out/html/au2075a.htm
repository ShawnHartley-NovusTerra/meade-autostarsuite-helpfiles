Edit User Data (Tools menu)

To Edit User Data in an Autostar II, Revision 1.3o or higher of the Autostar II Software is required.

This function will display a dialog box for viewing and editing User Information. Owner Info and Site Information may be viewed, edited, saved and retrieved for all handbox types. For the Autostar II handbox, Periodic Error Correction tables may also be saved and retrieved, but not edited.

When the Edit User Data menu item is selected, there will be a brief pause as the user data is retrieved from the handbox. Once the dialog displays, the user may edit the Personal Information fields on the left, and/or add/delete/edit the sites listed on the right.

To save a set of user data, use the Save button on the right. To retrieve an existing set of user data, use the Load button on the right. Files are stored with a ".usr" extension.

For Autostar II only, advanced users may wish to restore their Periodic Error Correction (PEC) data should they ever erase their user data from the handbox. Each time user data is saved to disk, a copy of the PEC Table for both axes is stored in that file. To restore such a saved PEC Table, load the ".usr" file by pressing the Load button, then check the "Include PEC Tables" check box prior to pressing the Send button. Doing so will erase any PEC, Personal Information and Site data already in the handbox.
