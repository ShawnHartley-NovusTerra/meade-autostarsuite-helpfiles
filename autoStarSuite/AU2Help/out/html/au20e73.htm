Send Data (Autostar Commands Button Group)

Use this command to copy the current contents of the Handbox List View to the Autostar handbox. Sending data to the handbox erases all the objects already in the handbox, so if you do not wish to permanently lose this data, Retrieve it first, Save it to the hard disk, and then proceed with the transfer.

Every time you Send Data or Retrieve Data from the Autostar Handbox, a backup file of the current Handbox List contents is saved to the hard disk.

To restore this list and erase any changes you have made since the backup file was created, choose Restore Data from the File menu.
