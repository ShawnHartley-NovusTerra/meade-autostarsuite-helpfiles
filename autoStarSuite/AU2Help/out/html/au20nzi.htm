How To Get Started Using Autostar Update

Autostar Update performs two main functions:

1)Loads new software into the Autostar Handbox

2)Transfers object menu choices such as comets, asteroids and user objects between the Autostar Handbox and the PC.

Before you can perform either of these functions, your Autostar Handbox must be properly connected to your Personal Computer. If you have not already done so, you must purchase the optional LX200 Interface Cable, #505 Connector Kit or #506 Connector Kit. Follow the instruction sheet that came with the connector kit to fully utilize the software. The Autostar Update can be used without connecting the Autostar handbox to the PC, but you will be limited to manipulating the data on the PC only.

Loading new software into the Autostar Handbox

The Autostar Update document window opens in the basic view. If all you wish to do is update the software in your handbox to the latest revision, connect your computer to the internet, connect your Autostar Handbox to your PC, and click on the Upgrade Autostar Software Now button.

Autostar Update then automatically checks the latest version of the software found on Meade’s Web Site (if you are connected to the internet), also checks your local hard disk to see if a version is already stored there, and finally checks your handbox.. The versions of the software that were found then display in a dialog box, including the revision level of the software in your handbox. Follow the prompts in the dialog boxes to complete the operation. Note that transferring this data to your handbox takes approximately 15 - 25 minutes. Caution: Do not disturb the handbox during this operation.

Transferring objects between the Autostar Handbox and the Personal Computer

To select this function, click on the Advanced View button. The window expands to show the Autostar Commands buttons, the Library List View and the Handbox List View. The Library List View displays a list of objects that are currently in Autostar Update’s memory. Use this list view to manipulate, sort and edit objects prior to transferring them to the Autostar Handbox. The Handbox List View displays a virtual list of objects contained in the Autostar Handbox. Build this list by transferring desired objects from the Library List View. Hint: Sort and select the objects you wish to transfer to the Autostar Handbox for your next observing session.

To load data into the Library List View, Open an existing Autostar Update Data (*.AUD) File, or Import certain file types for comet, asteroid, satellite and tour data (you may find some of these files on the Meade Instruments Web Site), or the program can access the data directly from the web by using File->Get Object Data from WWW

Use the Autostar Commands buttons to Connect to the Handbox, and Send Data and Retrieve Data.
