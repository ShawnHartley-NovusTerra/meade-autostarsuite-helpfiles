The Handbox List View

The Handbox List View contains the data that will be Sent to the Autostar Handbox, or that has just been Retrieved from the Autostar Handbox. Consider this view (on the right half of the screen) as your "Temporary Holding Area" for objects that are about to be Sent to the handbox. The order that the objects are displayed in the Handbox List View is the same order that they will appear in the Autostar Handbox. So prior to sending the data, click on the column headings to sort the data in a way that is meaningful to you, such as by Name, Right Ascension, Absolute Magnitude, etc.

Autostar allows manipulation of six types of objects: asteroids, comets, satellites, landmarks, user objects (which may be customized with the Autostar II) and tours. To select the object type you wish to view and/or manipulate, choose one of the six buttons located at the top of either list view.

You may Save the data in the handbox list and Load it into the handbox list at a later date.

Since sending data to the handbox will erase all the objects already in the handbox, if you do not wish to permanently lose this data, Retrieve it first, Save it to the hard disk, and then proceed with the transfer.

Whenever data is Sent to or Retrieved from the handbox, the contents of the handbox list are automatically saved to a backup file, so you can undo any changes made by choosing the File Menu Item Restore Data.
