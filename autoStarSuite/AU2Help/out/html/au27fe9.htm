Set Com Port(Options menu)

This will present a dialog box to allow you to set the COM port that the Autostar is connected to. You can select any of the 4 standard COM ports or enter a number beyond those by selecting "other" and then entering the number of the COM port in the Specify box. Only enter the number, don’t enter the COM before it. If you’re not sure which COM port the Autostar is connected to click on Auto-Detect and the updater will find it and set it for you. Auto-Detect will only check the 4 standard COM ports.
