How To Retrieve The Objects Stored In The Handbox

When you first launch Autostar Update, the right List View Window, representing the handbox data, will be empty. This window is a temporary storage place for building and organizing your handbox data list. Note: Once you click on the Send Data button, the contents of this window OVERWRITES the current objects contained in your handbox. Therefore, you may wish to retrieve the handbox data first, and then add to, edit, re-arrange and/or save that data before overwriting it.

To retrieve the asteroids, comets, satellites, landmarks, user objects and tours that are currently stored in the handbox:

1.Connect the Handbox to the PC using the appropriate cable kit.

2.Turn the telescope power on.

3.Click on the Retrieve Data button in the Autostar Commands button group located in the upper right of the Advanced View window. It may take several minutes to retrieve the data, depending on how much data is in the handbox. Once the data is retrieved, it is automatically saved to a backup file, so you can undo any changes you make before clicking on the Send Data button by choosing the File Menu Item Restore Data….
