Set Background (Options menu)

This will present a dialog box to allow you to set your own, custom image as the background of the Autostar Update application window. You may select any BMP, JPG, or GIF file. It will be stretched to fit the proportions of the application window, so it is a good idea to start with an image that is roughly the same proportions.

To restore to the background to the Rosette Nebula image that came with the application, choose the Reset Image item from the popup menu.
