Edit Object(s) (Edit menu)

Displays a new dialog box containing all the data for the selected object(s). You may only select objects from one list view at a time. To edit the data, select the field you wish to modify and type the new data in the edit box. To abort the change and restore the original values, click on Cancel.

Note: The data for many of the fields is limit-checked to ensure the integrity of the data. For example, the declination of a User Object is by definition an angle between –90 and +90 degrees, so a value of 150 will be rejected. Be careful when entering or modifying data.

Right-mouse-clicking a selected object(s) from either list view may also access this command.
