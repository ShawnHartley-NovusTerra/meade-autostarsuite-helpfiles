Get object data from WWW (File Menu)

Use this command to retrieve object data from the internet and have it added directly to the library. The object data comes from various sources on the web. The files available for download are determined dynamically when you run the Autostar Update, so you may notice different choices on different days. All of the following sites update their data regularly, so you may find it best to refresh your ephemerides data just before an observing session.

·Asteroidshttp://cfa-www.harvard.edu/iau/Ephemerides

·Cometshttp://cfa-www.harvard.edu/iau/Ephemerides/Comets/

·Satelliteshttp://celestrak.com/NORAD/elements/

·Tourshttp://www.meade.com/support/auto/AutostarTourFiles/

You can select individual body types to download or all the body types. If you select all body types more than 2000 objects will be added to your library.

Before starting this operation you should be connected to the Internet. If you have AOL, login to AOL and then come back to the update program.

A dialog box will be displayed allowing you to select which objects you wish to download. To select all objects of a given type, check the checkbox next to the category type, such as Comets or Satellites. To download individual items only, click the + next to a category to have a list of available download files generated dynamically by querying the applicable web servers. After selecting the objects to download, you have the following additional options to choose from:

·Delete all existing objects first

This will delete all the object data of the specified body type before adding new data

·Only import objects that already exist

This will effectively update the body data in your library by not adding new objects but only replacing the ones that match the names in your library.

After pressing OK look to the lower left corner status bar. There will be an indication of the progress of the download. When the operation is complete the Library List View will be updated with the new data.
