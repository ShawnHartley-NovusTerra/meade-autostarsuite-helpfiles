Options menu commands

The Options menu offers the following commands, which enable you to choose the default functions of the program:

Startup Connect->Connect Only Toggles option to automatically connect to the handbox the next time the program is launched.
Startup Connect->Connect & Retrieve Toggles option to automatically connect to the handbox and retrieve data the next time the program is launched.
Advanced Startup Toggles option to start the program in advanced mode.
Verify Downloads Toggles option to verify all data sent to the handbox.
Load Recent Toggles option to load the most recently used library document the next time the program is launched
Set Com Port Select the computer com port connected to the autostar.
Set Background->Choose Image… Selects a user-supplied image file to use as the background of the application.
Set Background->Reset Image… Restores the background to the original image that came with Autostar Update.
Max Baud Rate Specifies the maximum baud rate to use for serial communications with the handbox (Autostar II Only)
