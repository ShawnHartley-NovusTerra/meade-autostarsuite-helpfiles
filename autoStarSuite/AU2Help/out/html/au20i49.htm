Transfer Objects to Handbox (Dialog Box Button)

When this button is clicked, all selected objects in the Library List View are copied to the Handbox List View. If an object with the same name already exists in the Handbox List View, you are prompted to overwrite that object.

You may also transfer the selected objects by dragging and dropping from one list view to the other.

You may only transfer objects of one type at a time. For example, Asteroids or Satellites. When you switch object type, your selections of the previous object type are lost.

You may select multiple objects from a list view by clicking one object, holding down the <SHIFT> (sequential) key or <CTRL> (non-sequential) key, and selecting another object.
