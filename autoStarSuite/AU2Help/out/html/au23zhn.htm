Delete Object(s) (Edit menu)

Deletes the selected object(s) from one of the list views. Note: This command will not immediately delete data from the library file stored on the hard drive – you must Save the file to overwrite that data. It will also not immediately delete data from the handbox until you Send the modified handbox list to the handbox to overwrite that data.

Right-mouse-clicking a selected object(s) from either list view may also access this command.
