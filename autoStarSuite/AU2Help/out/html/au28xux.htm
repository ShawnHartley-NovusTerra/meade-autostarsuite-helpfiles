Main Dialog Box Buttons

The Main Dialog Box offers the following buttons and features:

Upgrade Autostar Software Now Upgrades the handbox software.
Advanced Functions Toggles between "Basic" and "Advanced" view.
Autostar Commands Commands for communicating with the handbox.
Transfer Buttons Copies objects from one list view to the other.
Handbox List View The list of objects contained in the handbox list.
Library List View The list of objects contained in the library list.
Status Bar Status and information.
List View Status Status information about the Library List View and the Handbox List View.
Exit Exits Autostar Update.
