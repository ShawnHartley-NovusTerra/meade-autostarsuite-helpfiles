The Library List View

The Library List View displays a list of objects that are currently in Autostar Update’s memory. Use this list view to manipulate, sort and edit objects prior to transferring them to the Autostar Handbox. Consider this view (on the left half of the screen) as your "Master Library" which you build by combining objects from numerous sources, such as the handbox, the www, or manually entered data. It is possible to store tens of thousands of objects in your Library, and to have several different libraries. However, since you could not possibly observe all of these objects in a single outing, and the handbox space is limited, you should selectively Transfer objects to the handbox collection, sort them into a logical order, and then Send that data to the handbox for use during your next observing session.

You may Open an existing library (*.AUD) file from a local disk drive, or you may create a New library document. Data can be added to a library manually through the "EDIT" - New command, by transferring data from the Handbox List View, or by Importing data from one of several different file formats.

Autostar allows manipulation of six types of objects: asteroids, comets, satellites, landmarks, user objects (which may be customized with the Autostar II) and tours. To select the object type you wish to view and/or manipulate, choose one of the six buttons located at the top of either list view.

Since amateur astronomy is such a popular pastime, there are numerous sources on the www where astronomical data can be obtained. Visit Meade Instruments Corporation’s web site for links to some of these resources.

Once your Library collection is built, you may Save or Save As to store that library for future use with Autostar and Autostar Update.
