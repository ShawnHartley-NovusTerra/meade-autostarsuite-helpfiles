Refresh Handbox Objects (Dialog Box Button)

When this button is clicked, all objects in the Library List View that already exist in the Handbox List View are copied to the Handbox list. Only the selected object type, as indicated by the depressed object button at the top of either list view, is affected.

Use this command to update ephemerid data in the handbox. For example, the handbox contains several satellite objects that you regularly observe, but new data for those objects has been obtained from the Internet. Rather than rebuild the handbox list, use this command to update the data in the existing list.
