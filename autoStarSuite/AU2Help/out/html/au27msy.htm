Retrieve Data (Autostar Commands Button Group)

Use this command to copy the current contents of the Autostar handbox’s database to the Handbox List View. Once the objects are copied over, you may Edit, Add to, Delete and Save the data, and then Send the modified objects back to the handbox.

Every time you Send Data or Retrieve Data from the Autostar Handbox, a backup file of the current Handbox List contents is saved to the hard disk.

To restore this list and erase any changes you have made since the backup file was created, choose Restore Data from the File menu.
