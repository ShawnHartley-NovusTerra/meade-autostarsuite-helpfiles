Stretch Command (Process menu)

This command is used to numerically alter the pixel values of an image. Several different stretches are available

LinearWhen selected a dialog will appear. The scale factor is the number that will be multiply each pixel value. The pixels below the Min Value or above the Max Value are replaced with the limits and then processed.

LogEach pixel is replaced by the log of its value.

ExponentialEach pixel is replaced by Exp(pixel).

GammaEach pixel is replaced by its square root.
