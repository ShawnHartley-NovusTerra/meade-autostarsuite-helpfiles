Group Menu commands

The Image menu offers the following commands:

Edit Allows you to define a group of image files to processed in a single batch
Copy Copies the current group of images to a new group of image files, each named by prepending “COPU_OF_” the start of each image file name.
Examine Allows you to cycle through display of all of the groups in the currently defined group.
Delete Deletes the currently defined group of images from the disk
Calibrate Performs calibration on all the images in the group using the same calibration file values.
Align Aligns all of the images in the group.
Combine Merges all of the images in the group together into a single image using a variety of methods.
Filter Applies one of several image enhancement filters to the entire group.
Normalize Linearly rescales all images in the group such that their average intensities are 10,000 ADUs.
Fix Column Fixed a defective column in all images in the group.
Photometry Performs aperture photometric measurements on all images in the group, reporting results in columnar form.
