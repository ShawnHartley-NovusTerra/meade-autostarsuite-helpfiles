Merge (Process menu)

The Merge Image command allows you to select an image from the Merge File to perform a Merge Function. You can select; Add, Subtract, Average, Median Combine, Multiply, Divide, Minimum, Maximum, And, Or, and Exclusive Or functions. There is also the option to select x/y offsets, and Scale Factors of the Image File and Merge File. (For a straight add or subtract both of these Scale Factor values should be set to 1.0).
