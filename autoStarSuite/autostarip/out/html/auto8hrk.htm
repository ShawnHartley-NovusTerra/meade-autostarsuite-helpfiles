Combine Command (Group menu)

When command is used to combine all of the images in the currently defined image group into a single output image which is displayed at the end of the operation. There are several different combination methods:

SumThis method adds all of image together.

AverageThis method averages all the images together.

MedianThis method uses the median of all the pixels at a given position in all of the images a the pixel at that location in the output image.

MINThis method takes the smallest of all the pixels values at each location.

MAXThis method takes the largest of all the pixel values at each location.

The resulting image becomes the current IP image and is untitled.
