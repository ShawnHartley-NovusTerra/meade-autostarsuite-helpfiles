View Menu commands

The View menu offers the following commands to manipulate the display of the image on the computer display. None of the commands under the View menu change the actual image data, they only alter the way it is displayed.:

Zoom Allows the apparent size of the image to be altered.
Scale Allow scaling the image in a variety of ways: Contrast, Brightness, Log, Exponential, Histogram Equalization, and Posterization.
Palettes Allows selection of various color palettes.
Contour Allows contour lines to be drawn over the current image.
Blink Images Allows two images to be compared by rapidly toggling the display between the two images.
