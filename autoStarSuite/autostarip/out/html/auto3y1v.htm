Photometry Command (Group menu)

This command allows you to perform the same photometric measurements on all images in the group. It is designed to ease the data analysis load when many obervations of the same field of view have been taken to measure a variable star, determine air mass, or track asteroids. When this command is selected, the following dialog appears:

When the dialog appears, draw boxes around each star or object you wish to measure. If the images have not been previously aligned, or if one of the targets, such as an asteroid or planet, are moving, you should check the Recenter and Track option. Each new image is recenter all of the measurement apertures in each successive image.

To facilitate easy import of the result data into a spread sheet, select the Columnar Format option.

If you make a mistake during star selection, clicking Clear List, will allow you to start over. When you are ready, click on Process and your measurements for each image will be written to the current log file.
