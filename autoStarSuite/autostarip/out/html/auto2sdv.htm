Zoom Image Size (View menu)

This command allows you to change the displayed size of the image. The image can be made larger or smaller depending on your needs. This is generally used to allow an entire image to be displayed even if the original image size exceeds the screen size.

The Fit Window button automatically scales the image so that all of the image can be displayed in the current window regardless of the aspect ratio of the window. This means that one axis of the image may not completely fill a portion of the window. You will notice that the Photometry cursor is constrained from using the area outside the actual image area.
