Preference Dialog (Options Menu)

The Preference dialog allow you to selectively enable or disable a number of features in the program.. The following tables describes the various fields:

The right half of the dialog controls image display options. The following controls are present:

ControlDescription

Auto ContrastWhen enabled, the program will automatically prescale images at load time using the method specified (Threshold, Std Dev, Min/Max). This makes it quick an easy to load an examine images. If you need to custom scale the image, you can then use the SCALE command or toolbar button to make fine adjustments.

Fit Window On LoadThis option, when selected will automatically, zoom the image in or out on load so that it completely fills the display window.

Group Operation OverwriteWhen selected, group operations that change the image file contents modify the files in the group. When this option is not set, a copy of the file is created with a name that shows the operations performed on the file..

24 Bit ColorModern PC’s us displays that can display more than 65000 color simulaneously. Setting this option causes the program to take advantage of this capability when load and creating color images. When unchecked, the program is compatible with older 8 bit color systems and will map created color images into palettized 8 bit images.

The right half of the Preference dialog allows you to enabled or disable fields on the status display bar at the bottom of the page.

FieldDescription

DateDisplay the current Gregorian date.

Local TimeDisplays the current local time or the current set time.

x,yDisplays the Right Ascension value of the mouse position when in the Starmap display, or the x and y coordinates when in an Image display.

DnDisplays the Declination value of the mouse position when in the Starmap display, or the pixel value under the cursor when in an Image display. The pixel value show both the 8 bit and 16 bit values simultaneously.

Dist.Displays the Altitude value of the mouse position when in the Starmap display, or the Distance value if the Set Distance Reference is enabled.

AngleDisplays the Azimuth value of the mouse position when in the Starmap display, or the Angle value if the Set Distance Reference is enabled in an Image display.

NOTE: If all of the fields are enabled, you may not see every field if your screen resolution is to low. Try selecting fewer fields or change to a higher screen resolution.
