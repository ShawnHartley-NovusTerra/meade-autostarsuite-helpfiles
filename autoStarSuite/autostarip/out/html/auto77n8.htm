Tools menu commands

The Tools menu contain commands that allow you to explore the information in and about your image. None of the Tool commands change your actual image data.

Magnifier Displays the Magnifier. This tool allows you to zoom into any area of the display.
Histogram Displays the Histogram for the current image.
Coordinates Displays the modeless Coordinates dialog box. Useful for viewing the image data in various formats.
Information Displays information about the current image.
