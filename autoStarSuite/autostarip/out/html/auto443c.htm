Process Menu commands

The Process menu contain commands that alter the actual image data. These commands are used to calibrate, enhance and process image data.

Undo Undoes the last image processing function.
Calibrate Allows selection of a Bias Frame, Dark Frame, and Flat Field Frame, then automatically performs all of the steps necessary to calibrate the CCD Image.
Merge Allows merging the currently opened image with another using a variety of functions, such as, Add, Subtract, And, Xor, Median Combine, etc.
Fix Column Defect Replaces defective imager columns with the average of adjacent columns.
Flip Image Flips the current image about either the vertical or horizontal axes.
Rotate Image Rotates the image 90 degrees clockwise or counter-clockwise.
Resample Changes the actual size of the image to make it larger or smaller
Convolution Filters Allows various convolution filters to be applied to the image: Smoothing, Sharpening, Edge Enhancement, Edge Only, etc..
Unsharp Masking Improves the details visible in an image by utilizing a process known as Unsharp Masking.
Median Filter Removes small defects from the image by passing a Median Filter over the data.
Fix Hot Pixels Removes individual bright or hot pixels from the image caused by cosmic rays or other noise sources.
Fix Cold Pixels Removes individual dim or pixels from the image caused by cosmic rays or other noise sources in calibration images.
Add Noise Adds uniform or gaussian noise to an image
Add Stretch Image Data Transforms image data by applying linear, geometric, or log functions to the pixel data.
RGB_Merge Allows full color images to be produced by merging three images. Each taken through a Red, Green, and Blue filters, respectively.
