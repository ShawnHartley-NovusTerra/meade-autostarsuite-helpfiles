Edit Command (Group menu)

This command is used to create and modify image groups. An image group is just a list of file names, stored one per line. When the command is selected the Group Edit dialog box appears similar to the one below:

To erase the current group contents, click on the Clear List button. To select additional files into the list using a windows browse tool, click Add to List. The browse box will allow you to select multiple files by using the Shift and Control key in connection with your mouse when clicking files. When you are satisfied with your selections, click on Open in the browse control. To edit the actual group list with a text editor, click on the Edit button.
