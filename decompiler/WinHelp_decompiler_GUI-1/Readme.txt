------------
 Overview:
------------

HlpDecoGUI is a GUI shell/wrapper of HELPDECO.

HELPDECO is a nice little program, which de-compile WIN-HELP files. But it is a console program, so sometimes it is very tiresome and error prone to use it correctly.

So I created this GUI shell for HELPDECO. [Version 2.1(32bit), available at the time of creation of this program.]

The newest public version of HELPDECO is always available at:
http://www.geocities.com/mwinterhoff/helpdeco.htm

The newest public version of HlpDecoGUI is always available at:
http://www.geocities.com/dipakauddy/hlpdecogui.htm


---------------- 
 Requirements:
----------------

HELPDECO de-compiler.


--------------
 How To Use:
--------------

For the first time you launch HlpDecoGUI, you will have to set the de-compiler path (i.e. path of HELPDECO.EXE), this need to be done only once, for next sessions HlpDecoGUI will remember the de-compiler path. (you can always re-configure the de-compiler path, just click on the 'Reset' button).

Once de-compiler path is set you can use HlpDecoGUI to de-compile WIN help files.

NOTE: Presently HlpDecoGUI only supports "/y, /m, /g, /b, /c, /d, /x" switches of HELPDECO.
Switch "/y" is necessary for HlpDecoGUI to perform correctly.

(For more details see documentations comes with HELPDECO)
